# Changelog

## v0.1.0 - 2026-06-29

Initial public release.

- Local private-network note board for trusted LAN or VPN use.
- Live browser updates for new notes and replies.
- Newest-first note layout capped at 50 visible notes.
- Create, edit, reply, and delete flows.
- Local SQLite persistence with backup, restore, and backup maintenance scripts.
- Optional sound and desktop alerts for incoming notes and replies.
- Five dark House Notes themes and local logo asset.
- Test/check scripts for local verification.
