# Security Policy

House Notes is intended for a trusted private LAN or VPN only. It has no user accounts by design, so anyone who
can reach the server can read, create, edit, and delete notes.

Do not expose the app directly to the public internet. Do not publish local databases, backups, logs, `.env`
files, or private note data.

Report sensitive issues privately to the repository owner. Do not open a public issue containing credentials,
private notes, database files, or live exploit details.
